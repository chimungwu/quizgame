# 📐 平方測驗 Square Quiz

一個用 **純前端 HTML + JavaScript** 製作的簡單小遊戲，  
讓小朋友可以練習指定範圍（1–100）內數字的平方運算。  
題目隨機產生，不重複，可選擇 **5、10、15 或 20 題** 來挑戰。

---

## 🚀 功能特色
- 自訂題目範圍（1–100）
- 題目隨機，不會重複
- 可選擇題數：5、10、15、20
- 自動批改，立即顯示正確 / 錯誤
- 簡單、快速，直接瀏覽器就能玩
